# 🎯 AI Student Exam Assistant

Plan smarter. Study better. Score higher.

## 🚀 Overview
AI-powered assistant that generates personalized study plans based on:
- Exam timeline
- Daily study hours
- Subject
- Preparation level

Built using **Streamlit + Google Gemini AI**

---

## 💡 Problem Statement
Students often:
- Don’t know what to study first
- Waste time on low-priority topics
- Lack structured planning

---

## ✅ Solution
This app:
- Analyzes student context
- Generates day-wise study plans
- Suggests priority topics
- Provides motivation strategies

---

## 🧠 Features
- 🎯 Personalized study plans
- ⚡ Smart strategy (Crash / Revision / Balanced)
- 📊 Dynamic input sliders
- 📥 Download study plan
- 🧠 AI-powered recommendations

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Google Gemini API
- Prompt Engineering
